# Jake Fieldhouse

Personal professional site for [jakefieldhouse.co.uk](https://jakefieldhouse.co.uk).

The site is intentionally small, fast, and public-source friendly: static Vite, no application framework, no generated build output committed, and no private working notes.

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Positioning

Jake Fieldhouse is presented as a technical operator for useful, commercially valuable work across AI workflows, automation, systems, hardware-aware troubleshooting, and pragmatic implementation.
